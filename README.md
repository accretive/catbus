Dependencies
------------
* Node
* Harp (and component)
* Bower
* harp/components/compass

Getting Started
---------------
1. clone: `git clone http://github.com/drbrts/catbus`
2. globals: `npm install -g harp component`.
3. go: `cd catbus`
4. compass: `cd catbus && component install harp/compass`
5. bower: `bower install`
6. run: `harp server`
